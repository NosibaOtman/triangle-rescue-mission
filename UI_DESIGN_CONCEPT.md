# משימת הצלת המשולשים - קונספט עיצוב UI/UX

## 1. פילוסופיית העיצוב

### תנועה עיצובית: "Modern Pedagogical Playfulness"
עיצוב שמשלב:
- **בהירות חינוכית** - כל אלמנט משרת מטרה לימודית
- **משחקיות מבוקרת** - אנימציות וצבעים שמעניינים אך לא מסיחים דעת
- **אנושיות** - עיצוב שמדבר לתלמידי כיתה ז' בשפתם

### עקרונות ליבה
1. **Clarity First** - כל דבר חייב להיות ברור ומובן
2. **Color as Information** - צבעים מעבירים משמעות (זוויות vs צלעות)
3. **Motion with Purpose** - כל אנימציה מוסיפה משמעות, לא רק קישוט
4. **Inclusive Design** - גם תלמידים חלשים וגם חזקים יכולים להצליח

## 6. רכיבים (Components)

### 1. TriangleVisualizer
**מטרה:** הצגת משולש עם סימונים ברורים

**אלמנטים:**
- SVG משולש בגודל 300x300px
- קשתות לזוויות (בצבע סגול, #8B5CF6)
- קווים עבים לצלעות (בצבע תכלת, #06B6D4)
- תוויות: זוויות בדרגות, צלעות באורכים יחסיים
- סימון קודקודים: עיגולים קטנים

**אנימציה:**
- כניסה: fade-in + scale (0.8 → 1.0)
- משך: 0.5 שניות

---

### 2. AnswerButton
**מטרה:** כפתור בחירה תשובה

**מצבים:**
- Default: כחול בהיר, shadow קל
- Hover: כחול כהה יותר, shadow גדול יותר
- Active: כחול כהיר, border עבה
- Disabled: אפור בהיר, cursor not-allowed

**טקסט:**
- Fredoka, 16px, Bold
- צבע: לבן
- Padding: 12px 24px

**אנימציה:**
- Hover: scale(1.05), shadow increase
- Click: scale(0.95), feedback immediate

---

### 3. ProgressBar
**מטרה:** הצגת התקדמות במשימה

**אלמנטים:**
- סרגל ליניארי בחלק העליון
- מספור: "שלב 3 מתוך 10"
- תרשים דלתות (10 דלתות, כל דלת = שלב)

**צבעים:**
- דלת סגורה: אפור בהיר
- דלת פתוחה: ירוק בהיר
- דלת נוכחית: כחול בהיר

**אנימציה:**
- דלת נפתחת: rotate(90deg) + fade-in
- משך: 0.5 שניות

---

### 4. FeedbackPanel
**מטרה:** הצגת משוב חינוכי

**מצבים:**
- Correct: רקע ירוק, אייקון ✓, טקסט בהיר
- Incorrect: רקע אדום, אייקון ✗, טקסט בהיר
- Hint: רקע כתום, אייקון 💡, טקסט בהיר

**אנימציה:**
- כניסה: slideUp + fade-in
- משך: 0.3 שניות

## 7. מערכת אנימציות

### Entrance Animations
```
Fade In: opacity 0 → 1 (300ms)
Slide Up: translateY(20px) → 0 (300ms)
Scale In: scale(0.9) → 1 (300ms)
Rotate In: rotate(-5deg) → 0 (300ms)
```

### Interaction Animations
```
Button Hover: scale(1.05) (150ms)
Button Click: scale(0.95) (100ms)
Shake: translateX(±5px) × 3 (200ms total)
Pulse: scale(1) → 1.05 → 1 (1000ms loop)
```

### Success Animations
```
Celebration: confetti-like particles (1000ms)
Door Opening: rotateY(90deg) (500ms)
Progress Fill: width 0 → 100% (500ms)
```

### Error Animations
```
Shake: translateX(±8px) × 3 (300ms)
Bounce: translateY(±5px) × 2 (200ms)
Pulse Red: background color pulse (500ms)
```

## 8. Responsive Design

### Breakpoints
```
Mobile: < 640px
Tablet: 640px - 1024px
Desktop: > 1024px
```

### Adjustments
**Mobile:**
- משולש: 200x200px (במקום 300x300px)
- כפתורים: 2 שורות, 2 כפתורים בשורה (בשלבים מסוימים)
- טקסט: קטן יותר (14px במקום 16px)
- Padding: sm (8px) במקום md (16px)

**Tablet:**
- משולש: 250x250px
- כפתורים: 3 בשורה
- טקסט: רגיל (16px)

**Desktop:**
- משולש: 300x300px
- כפתורים: 3 בשורה
- טקסט: רגיל (16px)

## 9. Accessibility (נגישות)

### צבעים
- Contrast ratio: לפחות 4.5:1 לטקסט רגיל
- לא להסתמך על צבע בלבד - להשתמש גם בסימונים (למשל: ✓, ✗)

### Keyboard Navigation
- Tab order: משמאל לימין, מלמעלה למטה
- Focus indicator: בולט וברור (outline כחול)
- Enter/Space: להפעיל כפתורים

### Screen Readers
- alt text לתמונות SVG
- ARIA labels לכפתורים
- Role attributes לרכיבים אינטראקטיביים

## 13. Implementation Notes

```

### Framer Motion
```
Fade In: initial={{ opacity: 0 }} animate={{ opacity: 1 }}
Slide Up: initial={{ y: 20 }} animate={{ y: 0 }}
Scale: initial={{ scale: 0.9 }} animate={{ scale: 1 }}
```

### SVG Styling
```
Angles: stroke="#8B5CF6" stroke-width="2"
Sides: stroke="#06B6D4" stroke-width="3"
